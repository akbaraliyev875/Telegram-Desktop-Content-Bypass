## 🚩 Loyiha haqida
Ushbu repozitoriya Telegram Desktop ilovasida **"Restrict Saving Content"** (Kontentni saqlashni taqiqlash) funksiyasi qanchalik zaif ekanligini isbotlash uchun yaratildi. 

Men Telegram jamoasiga ushbu zaiflik haqida **2 marta** batafsil hisobot (report) yubordim. Ammo ular buni "xato emas" deb hisoblashdi va munosib mukofotni (**Bug Bounty**) berishdan bosh tortishdi. 

**Mening javobim:** Agar mehnat qadrlanmasa, haqiqat hamma uchun ochiq e'lon qilinadi.

## 🛠 Texnik tahlil
Telegram kontentni interfeys darajasida cheklaydi, biroq fayl operatsion tizimning **Cache** (kesh) jildiga tushganidan so'ng, uni tizim imkoniyatlari orqali nusxalab olish (Copy/Paste) mumkin. 

**Metod:** `Show in Folder` -> `OS Explorer` -> `Bypass Successful`.
